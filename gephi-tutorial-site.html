Announcement: Gephi tutorial site in chinese is built

So, the site is built for a lab event in a workshop in 9/22. The reason I built this is because I think these basic guide might be needed in the begining of using this tool.

The website is served using the lab server: Gephi tutorial site

In addition, a webpage is built with the gephi plugin -- Sigmajs Exporter -- global migration network .

The website is built with a static-site generator, called Mkdocs, and they have the readthedocs template built-in.

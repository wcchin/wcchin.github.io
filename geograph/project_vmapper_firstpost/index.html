Monday, January 11, 2016 » geography, graph, GUI, map, tool

vmapper: a vector mapping tool

I’ve been working on a project for mapping the geographical data and viewing with browser. Using svg, scallable vector graph, as the main output file, this python library is written to read spatial data (currently shapefile, planned to get data from postgis database), and export them into svg file, with some simple mapping function (classification with qualitative data column, and planned to draw with sequential data column data).

Everything is still in ‘first attemp’ and planning phases. So I will keep working on this in following times.
